# 📂 Data Folder

## 📌 Description

This folder contains all datasets used in the analysis and modeling of fiber-optic network faults.

---

## 🗂️ Structure

* `raw/` → Original datasets (unaltered)
* `processed/` → Cleaned and prepared datasets for analysis

---

## 📊 Data Sources

* **TTCL Fault Records** (CSV)
* **Network Infrastructure Data** (GeoJSON)
* **Elevation Data (SRTM)**
* **Rainfall, Soil Moisture, Temperature** (TerraClimate)

---

## ⚙️ Notes

* Raw data should NOT be modified directly
* All preprocessing is done in the `/scripts/` folder
* Some datasets are not publicly available due to privacy restrictions (TTCL data)

---

## ⚠️ Disclaimer

Sensitive or proprietary data has been excluded or anonymized in this repository.
