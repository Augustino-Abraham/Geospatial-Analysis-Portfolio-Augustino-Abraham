# 🗺️ Maps Folder

## 📌 Description

This folder contains all GIS-generated maps produced in this project.

---

## 🗺️ Maps Included

* Fault Risk Map
* Terrain Slope Map
* Buffer Analysis Map
* Predicted vs Actual Fault Locations

---

## 🎯 Purpose

* Visualize spatial distribution of faults
* Identify high-risk zones
* Support infrastructure planning

---

## 🛠️ Tools Used

* QGIS
* ArcGIS

---

## 💡 Notes

All maps are exported in high resolution for presentation and reporting purposes.
