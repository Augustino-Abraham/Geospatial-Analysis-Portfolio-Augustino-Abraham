# 📂 Outputs Folder

## 📌 Description

This folder contains all generated outputs from the analysis and modeling process.

---

## 🗂️ Structure

* `maps/` → GIS-based maps (fault risk map, terrain maps)
* `figures/` → Charts, plots, and statistical visualizations

## 📊 Outputs Included

* Fault Risk Map
* Terrain Analysis Maps
* Correlation Plots
* ROC Curve & AUC Results

These outputs demonstrate:

* Spatial patterns of network faults
* Model performance and accuracy
* High-risk zones for proactive maintenance
